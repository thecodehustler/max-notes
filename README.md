## 食用方法

下载、安装 [Git](https://git-scm.com/downloads) 程序。

（Windows 10）在任意目录，按住 Shift+右键单击，选择 `Open PowerShell window here`。

在弹出的窗口中输入：

    git clone https://github.com/thecodehustler/max-notes

即可将整个项目同步到本地。
